Why Docker?

  • Reproducible builds
  • Ease of deployment
  • Isolation of individual devices (with multiple devices on single host)
  • Run across heterogeneous driver/toolkit environments
  • Requires only the NVIDIA driver to be installed
  • Enables "fire and forget" GPU applications

How ?

Specialized Docker images by NVIDIA https://github.com/NVIDIA/nvidia-docker/

Built on top of:

  • Ubuntu 16.04 LTS
  • Ubuntu 14.04 LTS
  • CentOS 6
  • CentOS 7

Usage

Easy as running: /usr/bin/nvidia-docker run -it tensorflow/tensorflow:latest-gpu

How to let a non-admin user use this

  1. Add non-privileged UNIX user on server
  2. that is not a member of the docker group
  3. have a long-running TensorFlow container
  4. Allow the non-privileged user to run
    docker exec -it ...
    (E.g. using sudo)
